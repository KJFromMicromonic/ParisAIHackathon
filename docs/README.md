# LiveKit Agents Documentation

This folder contains documentation for LiveKit Agents related to various integrations and best practices.

## Documentation Files

### [Google Realtime Models (Gemini Live API)](./google-realtime-models.md)
Complete guide for integrating Google's Gemini Live API with LiveKit Agents. Includes installation, authentication, usage examples, parameters, turn detection, thinking mode, and usage with separate TTS.

### [ElevenLabs TTS Plugin](./elevenlabs.md)
Documentation for using ElevenLabs as a Text-to-Speech provider in LiveKit Agents. Covers installation, authentication, usage, parameters, pronunciation customization, and transcription timing.

### [Mistral Voxtral STT Plugin](./mistral-voxtral-stt.md)
Guide for using Mistral's Voxtral model as a Speech-to-Text provider. Includes installation, authentication, usage examples, and available parameters.

### [Worker Entrypoint Best Practices](./worker-entrypoint-best-practices.md)
Comprehensive guide on worker lifecycle, entrypoint functions, request handlers, prewarm functions, worker load management, permissions, and best practices for building robust LiveKit Agents.

### [Realtime Models Overview](./realtime-models-overview.md)
General overview of realtime model plugins, including how to use them, available providers, considerations, and limitations.

### [MCP Servers with LiveKit Agents](./mcp-servers-with-livekit-agents.md)
Complete guide for integrating Model Context Protocol (MCP) servers with LiveKit Agents. Includes installation, usage examples, authentication, tool definitions, and best practices for building extensible agent systems.

### [LiveKit JavaScript Client SDK](./livekit-javascript-client.md)
Comprehensive documentation for the LiveKit JavaScript/TypeScript client SDK. Covers installation, connecting to rooms, publishing/subscribing tracks, device management, RPC, events, and browser support.

### [Building Accessibility App](./building-accessibility-app.md)
Step-by-step guide for building a voice-based accessibility application for blind and visually impaired users. Includes architecture overview, agent setup, MCP server examples (Google Maps, Airbnb), JavaScript client implementation, and best practices.

### [LiveKit Agents Vision](./livekit-agents-vision.md)
Complete guide for adding visual understanding to your LiveKit agents. Covers adding images to chat context, uploading images from frontend, sampling video frames, live video input with realtime models, video frame encoding, and inference detail settings.
